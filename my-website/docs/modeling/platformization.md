# Платформизация

## Push-уведомления

### Выбранное решение

**Firebase Cloud Messaging (FCM)** – сервис Google для отправки push-уведомлений на Android, iOS и веб.

### Почему выбрано FCM

- Бесплатное пользование
- Поддерживает все платформы
- Легко интегрируется с любым бэкендом через HTTP-API

### Сценарии отправки уведомлений

| Событие | Уведомление |
|---------|--------------|
| Организатор подтвердил заявку | "Ваша заявка подтверждена" |
| Организатор отклонил заявку | "Ваша заявка отклонена" |
| Статус игры изменился (открыта / заполнена / отменена) | "Статус игры изменён на ..." |

## Стратегия монетизации

### Бесплатная версия (с рекламой)

- Просмотр и поиск игр
- Запись на игры
- Базовый профиль
- Присутствует реклама

### Платная подписка (Premium, без рекламы)

- Отключение всей рекламы
- Приоритет в поиске (игры организатора видны раньше)

### Дополнительные перспективы

Если Sportnect станет реально популярным – продажа акций проекта.

## Модель бизнеса

**B2C (Business-to-Consumer)**

Приложение продаётся напрямую конечным пользователям (игрокам и организаторам), а не через бизнесы.

Любой человек может:
- Скачать / открыть приложение
- Пользоваться бесплатно с рекламой
- Оформить подписку без рекламы
