# Sportnect

**Информационная система для поиска и бронирования мест в сборных любительских играх по волейболу**

---

## Общая концепция

Sportnect - информационная система для поиска и бронирования мест в сборных любительских играх по волейболу. Это приложение, где пользователь может найти игру по своему уровню в удобном местоположении и записаться на неё.

---

## Основные функции продукта

- Регистрация и вход по номеру телефона

- Поиск по фильтрам (дата, время, цена (если корт платный), уровень игры, амплуа, сколько участников не хватает)

- Просмотр событий на карте (фото площадки и все критерии, которые были в пункте: "поиск по фильтрам")

- Чат с организатором и участниками игры в определённом событии

- Подтверждение или отклонение заявок организатором

- Личный кабинет (история игр, активные записи, избранные корты)
